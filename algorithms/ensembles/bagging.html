Bagging

Bootstrap AGGregatING — train the same model type on many different bootstrap samples of your data, then average their predictions. Bagging reduces variance without changing bias. Random Forest is bagging applied to decision trees with an extra feature-randomness twist.

Bootstrap sampling — why different data creates diversity

Each model trains on a bootstrap sample: draw N points from N with replacement. ~63.2% unique points appear. The remaining ~36.8% (out-of-bag) are a free validation set.

Variance reduction — why averaging helps

A single deep tree has high variance — it changes a lot with different training data. Averaging many trees cancels out the individual variation. The formula: Var(mean of B models) = σ²/B for uncorrelated models.

💡

Why does averaging reduce variance? If each tree has variance σ², the mean of B uncorrelated trees has variance σ²/B. With 100 trees, variance drops 100x. In practice trees are correlated (same dataset) so the reduction is smaller — which is why Random Forest adds feature randomness to reduce correlation further.

Common mistakes

✕
Bagging low-variance models. Bagging a logistic regression gives almost no benefit — LR already has low variance. Bagging works best with high-variance models like decision trees. If you bag a low-variance model, you waste compute for nothing.
✕
Not using oob_score=True. Set oob_score=True for free cross-validation. It’s nearly as accurate as 5-fold CV at zero extra training cost.
✕
Confusing BaggingClassifier with Random Forest. Random Forest = bagging + feature randomness at each split node. BaggingClassifier with max_features<1 does feature sampling per tree, not per node. These are different and Random Forest is usually better.

★
Key takeaways

Summary
Bagging trains the same model type on B bootstrap samples and averages predictions. It reduces variance without changing bias.
Works best on high-variance base models (deep trees). Useless on low-variance models (linear regression already stable).
OOB score is a free cross-validation estimate. Set oob_score=True always.
Var(average of B models) = σ²/B when models are uncorrelated. Random Forest reduces correlation further via feature randomness at each node.
For trees specifically, use Random Forest (sklearn) rather than BaggingClassifier — RF has additional feature-node-level randomness that typically helps.
