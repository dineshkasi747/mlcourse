Common mistakes

x
Trusting Pearson r for non-linear relationships. Load the "Curve" pattern — a clear U-shape gives r≈0. Always plot first. Use Spearman for monotonic non-linear relationships.
x
Correlation = causation. High r between ice cream sales and drowning deaths is a real example. Both are caused by summer heat. Always ask: is there a plausible causal mechanism?
x
Not checking for outliers before correlation. A single outlier can make r go from 0.1 to 0.8. Always scatter-plot your data and check Cook's distance for influential points.
x
Using Pearson on ordinal data. Rating 1-5 or Likert scales are ordinal, not interval. Use Spearman or Kendall's tau, not Pearson.
x
Chi-squared on small expected counts. Chi-squared is unreliable when any expected cell count is below 5. Use Fisher's exact test for 2x2 tables with small samples.

*
Key takeaways

Summary
Always scatter-plot numerical pairs before computing correlation. Anscombe's quartet proves why.
Pearson r = linear correlation only. Use Spearman rho for monotonic non-linear or ordinal data.
For numerical vs categorical: box plots + ANOVA F-test + effect size (Cohen's d or eta-squared).
For categorical vs categorical: contingency table + chi-squared test + Cramer's V for effect size.
Correlation is not causation. Always check for confounding variables and plausible mechanisms.
A single outlier can dramatically change Pearson r. Robust alternatives: Spearman, Kendall, or winsorised Pearson.
